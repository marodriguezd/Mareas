# 🌊 Costa Chipiona - Tabla de Mareas

**🔗 Sitio web en vivo:** [https://marodriguezd.github.io/Mareas/](https://marodriguezd.github.io/Mareas/)

Aplicación web interactiva para visualizar las mareas y el pronóstico marítimo en tiempo real para la costa de Chipiona (Cádiz), optimizada para visitas al Monumento Natural de los Corrales de Pesca.

## 🛠️ Tecnologías Utilizadas

- **Framework**: React 19 + Vite 6
- **Estilos**: Tailwind CSS v4 (con tipografía Outfit de Google Fonts)
- **Iconos**: Lucide React
- **Fuentes de datos**: API meteorológica y marina de Open-Meteo

## 🚀 Despliegue en GitHub Pages

Este proyecto se despliega directamente desde la carpeta `/docs` de la rama `main` en GitHub Pages.

### Desarrollo Local y Compilación:

1. Instalar dependencias:
   ```bash
   npm install
   ```
2. Servidor de desarrollo:
   ```bash
   npm run dev
   ```
3. Compilar para producción (actualiza la carpeta `/docs` para subir a GitHub Pages):
   ```bash
   npm run build
   ```
