# 🌊 Costa Chipiona - Tabla de Mareas

**🔗 Sitio web en vivo:** [https://marodriguezd.github.io/Mareas/](https://marodriguezd.github.io/Mareas/)

Aplicación web interactiva para visualizar las mareas y el pronóstico marítimo en tiempo real para la costa de Chipiona (Cádiz), optimizada para visitas al Monumento Natural de los Corrales de Pesca.

## 🛠️ Tecnologías Utilizadas

- **Framework**: React 19 + Vite 6
- **Estilos**: Tailwind CSS v4 (con tipografía Outfit de Google Fonts)
- **Iconos**: Lucide React
- **Fuentes de datos**: API meteorológica y marina de Open-Meteo

## 🚀 Despliegue en GitHub Pages

Este proyecto está configurado con **GitHub Actions** para compilar y desplegar automáticamente la aplicación en GitHub Pages con cada commit enviado a la rama `main`.

La URL de la publicación será: [https://marodriguezd.github.io/Mareas/](https://marodriguezd.github.io/Mareas/)

## 💻 Desarrollo Local

Para ejecutar la aplicación localmente en tu máquina o en Termux, sigue estos pasos:

### 1. Instalar dependencias
```bash
npm install
```

### 2. Iniciar el servidor de desarrollo
```bash
npm run dev
```

### 3. Compilar para producción
```bash
npm run build
```

Las compilaciones locales producirán un directorio `dist/` estático optimizado y listo para ser servido por cualquier servidor HTTP.
